# Repaso Linux - Álvaro

Este repositorio contiene los ejercicios de repaso de Linux organizados por bloques:

- **bloque1** → Ejercicios del capítulo 2 de Linux.
- **bloque2** → Trabajo en local con GIT e imágenes asociadas.

Cada bloque está en su propia carpeta y contiene los archivos `.md` y las imágenes necesarias.

## Estructura del repositorio
repasolinux_alvaro/
│
├── bloque1/
│ └── Ejercicios Linux- capítulo 2.md
│
├── bloque2/
│ ├── Trabajo en local-GIT.md
│ ├── image1.png
│ └── image2.png
│
└── README.md


- **bloque1:** contiene los ejercicios correspondientes al **capítulo 2 de Linux**, centrados en el uso básico de comandos y manejo del sistema.  
- **bloque2:** incluye el trabajo **"Trabajo en local - GIT"**, donde se practican comandos de Git en entorno local y la gestión de imágenes relacionadas con el ejercicio.

---

## 🧰 Descripción del trabajo realizado

1. Se creó un repositorio local llamado `repasolinux_alvaro` con la estructura anterior.  
2. Se añadieron los archivos Markdown (`.md`) y las imágenes correspondientes a cada bloque.  
3. Se realizaron **commits locales** para cada bloque.  
4. Cada bloque se trabajó en su propia **rama** (`bloque1`, `bloque2`) y luego se fusionó con la rama principal `main` mediante **merge commit**.  
5. Se generó este archivo `README.md` con información sobre el repositorio.  
6. Finalmente, se subió todo al repositorio remoto en **GitHub**, que es público.

---

## 🌿 Ramas utilizadas

- `main` → rama principal del proyecto  
- `bloque1` → rama para los ejercicios del capítulo 2  
- `bloque2` → rama para el trabajo en local con Git  

Cada rama fue creada, trabajada y fusionada con `main` una vez completado su contenido.
